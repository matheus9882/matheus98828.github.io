# Projeto-programacao-nova-ideia
O novo projeto consiste em criar um espaço para estabelecer grupo de pessoas que gostam do anime "pokémon" para poder competir entre si com a sua própria equipe. Contendo elementos de login e podendo escolher seus pokémons, salvar sua equipe, ver a equipe de outros jogadores etc. Também será possível alterar a sua equipe e modificar informações.
